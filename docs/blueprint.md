# **App Name**: GoldSight Analytics

## Core Features:

- Historical Data Viewer: Display XAUUSD historical price data starting from 2008.
- Price Forecast Generator: Generate XAUUSD price forecasts up to 2026 using generative AI. The tool uses the historical data to provide projections.
- Interactive Charting: Visualize XAUUSD trends with interactive charts that allow users to zoom and pan.
- Model Saving: Allow users to save custom forecast models for later use.

## Style Guidelines:

- Scheme: dark.
- Primary color: Gold (#D4AF37) for highlighting key financial metrics and elements. The hue is inspired by XAU, the ISO currency code for Gold.
- Background color: Deep blue (#191970) for the app's background, providing contrast. The desaturated color of a similar hue provides calm neutrality and lets the primary color stand out.
- Accent color: Lighter Gold (#FFD700) as a vibrant accent to draw attention to interactive elements and important updates, differing in brightness and saturation to make a great contrast to the primary gold color.
- Body and headline font: 'Inter', a sans-serif with a modern and neutral look suitable for financial data.
- Use clean and crisp icons related to finance and data visualization.
- Implement a responsive layout that adapts to different screen sizes.
- Use subtle transitions and animations to enhance the user experience.
